Bem vindos

A Finalidade desse site é mostrar uma abordagem mais prática dos conteúdos de programação, visto que em aula, há maior ênfase em conceitos e algumas vezes não em sua aplicação por diveros motivos.

É importante que o aluno pratique programação, e use o site apenas para o auxílio na hora de estudar os conteúdos.

Aqui serão encontradas as matérias de programação que a Escola Estadual de Ensino Médio Professora Maria Rocha disponibiliza no ensino médio integrado em informática. Estas matérias são:

  • Algoritmo e Programação;
  • Banco de Dados;
  • Desenvolvimento de Sistemas;
  • Programação Para Web 1;
  • Programação Para Web 2.
